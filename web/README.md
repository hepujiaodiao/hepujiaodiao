# 合浦角雕非遗介绍网站

## 网站概述

这是一个展示合浦角雕非物质文化遗产的网站，包含角雕介绍、传承代表人、制作过程、民间作品等内容。

## 网站结构

```
web/
├── index.html              # 首页
├── pages/                  # 页面文件夹
│   ├── introduction.html   # 角雕介绍页面
│   ├── inheritors.html    # 传承代表人页面
│   ├── process.html       # 制作过程页面
│   └── works.html         # 民间作品页面
├── components/             # 公共组件文件夹
│   ├── navbar.html        # 导航栏组件
│   └── footer.html        # 页脚组件
├── css/                    # 样式文件夹
│   ├── style.css          # 主要样式
│   ├── carousel.css       # 轮播图样式
│   ├── pages.css          # 页面专用样式
│   └── gallery.css        # 画廊样式
├── js/                     # JavaScript文件夹
│   ├── main.js            # 主要功能
│   ├── carousel.js        # 轮播图功能
│   └── gallery.js         # 画廊功能
└── images/                 # 图片文件夹（需要创建）
```

## 图片要求

1. **格式**：建议使用JPG格式，文件大小控制在500KB-2MB之间
2. **质量**：图片清晰，色彩真实，能够展现角雕艺术的精美
3. **内容**：图片内容要与页面主题相符，体现合浦角雕的特色
4. **版权**：确保使用的图片具有合法使用权
